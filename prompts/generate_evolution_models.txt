### Regles principals
1. El parent model és la base: no redisseny total.
2. Canvis incrementals, justificats i fàcils d'atribuir.
3. Evita repetir patrons de la família sense variació real.
4. Explica clarament què canvies respecte al parent, què vols provar i quines serien les següents variants.

### Guardrails d'arquitectura
- `architecture_definition` obligatori.
- `used_inputs` obligatori i no buit.
- `branches` obligatori i no buit.
- `output_heads` obligatori i no buit.
- `merges` opcional, però si existeix ha de ser llista.
- Cada item de `merges` ha de tenir `name`, `type`, `source_feature_maps` (llista no buida) i `output_feature_map_name`.
- No inventis inputs/targets fora dels disponibles.
- Mantén coherència de `maps_to_target_config_name`.
- `Add`/`Multiply`/`Concatenate`: `input_source_feature_maps` >=2.
- `AttentionKeras`/`MultiHeadAttentionKeras`: rank >=3.
- `TimeDistributed`: inner layer suportada.
- `Bidirectional`: requereix `wrapped_layer` o `inner_layer`.
- `Dense` requereix `units`; `Conv1D` requereix `kernel_size`; `GRU/LSTM` requereixen `units`.

### Claus prohibides (dialecte legacy)
No utilitzis aquestes claus dins `architecture_definition`:
- `inputs`
- `input_layers`
- `shared_backbone`
- `layers` a nivell arrel de `architecture_definition`

### Mini schema obligatori (V3)
`model_definition_full.architecture_definition` ha de tenir aquest patró:
```json
{
  "used_inputs": [
    {
      "input_layer_name": "...",
      "source_feature_name": "...",
      "shape": [800]
    }
  ],
  "branches": [
    {
      "name": "...",
      "input_source_layer": "...",
      "layers": [
        {"type": "Dense", "units": 64, "activation": "relu", "name": "..."}
      ],
      "output_feature_map_name": "..."
    }
  ],
  "merges": [],
  "output_heads": [
    {
      "output_layer_name": "...",
      "maps_to_target_config_name": "...",
      "source_feature_map": "...",
      "units": 1,
      "activation": "linear"
    }
  ]
}
```

Si utilitzes `merges`, cada merge ha de seguir aquest format mínim:
```json
{
  "name": "fusion_main_aux",
  "type": "concatenate",
  "source_feature_maps": ["main_features", "aux_features"],
  "layers_after_merge": [
    {"type": "Dense", "units": 64, "activation": "relu", "name": "fusion_dense"}
  ],
  "output_feature_map_name": "shared_features"
}
```
